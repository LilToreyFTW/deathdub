## 🚀 **Quick Start**

### **Linux:**
```bash
tar -xzf RegenerativeAddressesPro_v3.0_Linux.tar.gz
cd RegenerativeAddressesPro_Linux
./run_linux.sh
```

### **Windows:**
1. Extract `RegenerativeAddressesPro_v3.0_Windows_Package.zip`
2. Install Python 3.9+
3. Run `Run_RegenerativeAddressesPro_v3.bat`

### **Web Interface:**
- Access: http://localhost:8000
- Default Login: `admin` / `admin`
