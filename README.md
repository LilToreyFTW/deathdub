### **Linux:**
```bash

unzip RegenerativeAddressesToolPro-v3.1-linux.zip
```

### **Windows:**
1. Extract `RegenerativeAddressesToolPro-v3.1-windows.zip`
2. Install Python 3.10+
3. Run `RegenerativeAddressesToolPro.exe` or `run.bat`
